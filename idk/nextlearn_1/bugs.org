* B1: Prev/Next buttons appear on middle of screen

** Location
- Views/MainWindow.axaml (learning view layout)

** Root Cause
- DockPanel doesn't allocate space correctly when the nested Grid
  has proportional rows. Bottom bar ends up in the middle.

** Fix
1. Replace outer DockPanel with ~Grid RowDefinitions="Auto,*,Auto"~
2. Row 0: top bar | Row 1: content | Row 2: bottom bar

* FIXED B20: Teal box shows no content on first load (AvaloniaEdit)

** Location
- ~Views/MainWindow.axaml.cs~ (~OnLoaded~)

** Root Cause
- ~OnLoaded~ subscribed to ~PropertyChanged~ event for ~RenderedContent~
  *after* the initial value was already set. ~UpdateEditor()~ was never
  called for the first page, leaving the ~TextEditor~ empty.

** Fix
- Added ~UpdateEditor();~ call at the end of ~OnLoaded~ to pull the
  current ~RenderedContent~ value right after subscribing to future
  changes. Subsequent page navigations continue to update via the
  ~PropertyChanged~ handler.
