#+TITLE: MindmapDotnet
#+DESCRIPTION: Keyboard-friendly mindmap application — built with Avalonia + .NET

* Features
  - Keyboard-first: ~i~ (edit), ~Tab~ (child), ~Enter~ (sibling), ~Esc~ (done)
  - Arrow key navigation across tree nodes
  - Two view modes: Mindmap canvas + raw Markdown
  - Two layouts: Left-to-Right (horizontal tree) & Radial (center-out)
  - Save/load as plain Markdown files (GFM nested bullets)
  - Full Undo/Redo (Ctrl+Z / Ctrl+Y)
  - Node colors, collapse/expand subtrees
  - Search nodes (Ctrl+F)
  - Zoom (Ctrl+Scroll) + Pan (drag canvas)
  - Export to PNG
  - Light/Dark theme

* Save Format
  Mindmaps are saved as plain Markdown files with GFM nested bullet lists:

  ,* Root idea
  ,  * Child idea 1
  ,    * Grandchild idea
  ,  * Child idea 2

  Indentation depth maps to tree depth. The file is human-readable in any text editor.

* Keyboard Reference
  | Key              | Action                    |
  |------------------+---------------------------|
  | ~i~              | Edit selected node text   |
  | ~Tab~            | Create child node         |
  | ~Enter~          | Create sibling node       |
  | ~Esc~            | Exit edit mode            |
  | ~Delete~         | Delete selected node      |
  | ~←~ / ~→~        | Move to parent / first child |
  | ~↑~ / ~↓~        | Previous / next sibling   |
  | ~Ctrl+Z~         | Undo                      |
  | ~Ctrl+Y~         | Redo                      |
  | ~Ctrl+F~         | Search nodes              |
  | ~Ctrl+1-5~       | Assign color preset       |
  | ~Ctrl+L~         | Toggle layout (LTR/Radial)|
  | ~Ctrl+Scroll~    | Zoom in/out               |
  | ~Space+D~        | Toggle dark/light theme   |

* Layouts
  - **Left-to-Right**: Compact horizontal tree, root on left, children extend right
  - **Radial**: Classic mindmap style, root at center, children in concentric rings
  - Switch via toolbar button or ~Ctrl+L~
