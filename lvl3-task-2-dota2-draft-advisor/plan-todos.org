* DONE Phase 3: Scoring Engine — TPL & Race Condition

** 3.1 DraftScorer
[x] Create Services/DraftScorer.cs:
     - Method: ScoreAsync(DraftInput input) → List<DraftScore>
     - Filter out already-picked heroes (enemy + ally)
     - For each candidate hero ~(124 - 9 = 115):
         * Fetch matchups (already loaded in Phase 2)
         * Compute avg win rate vs 5 enemies
         * Apply Bayesian shrinkage: (wins + 50) / (games + 100) → pulls low-sample toward 50%
         * Role coverage score:
             - Check which roles are missing in ally team
             - Bonus if candidate fills a missing role
         * Lane fit score:
             - Check which lanes are open
             - Bonus if candidate's primary role fits an open lane
     - Weighted final: winRate*0.55 + roleCoverage*0.25 + laneFit*0.20
     - Normalize to 0-100 scale
     - Sort descending by score

** 3.2 Race condition demo
[x] Add RaceConditionSection.cs to Demo/:
     - First pass (BROKEN):
         * Use plain Dictionary<int, DraftScore>
         * Parallel.ForEach adds entries in parallel
         * No lock on counter → corrupted game count
         * Detect corruption by counting entries vs expected
     - Print corrupted results (red/yellow output)
     - Second pass (FIXED):
