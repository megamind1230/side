* DONE Wiki-link + pinned/archived fallback + deck-link prompt UI

** What
Wiki-links (~[[target]]~) now resolve pinned (~+target.md~) and archived
(~target.md~~) files when an exact match is not found. Cross-format extension
fallback (.md → .org) applies to all formats. All wiki-link variants work in

prompt.

** TryGetDeckLink fallback order
1. Exact ~target.md~ / ~target.org~
2. ~+target.md~ / ~+target.org~ (pinned)
3. ~target.md~~ / ~target.org~~ (archived)
4. Cross-format: if target has no extension, try .md → .org
